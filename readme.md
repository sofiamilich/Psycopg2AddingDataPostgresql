# Psycopg2AddingDataPostgresql

## Обзор проекта

`Psycopg2AddingDataPostgresql` — Данный скрипт предназначен для подключения к базе данных PostgreSQL, создания таблицы list_users и вставки данных в нее.


### Требования

Модуль psycopg2 должен быть установлен и доступен.



## Конфигурация

Конфигурационные переменные хранятся в файле config_db.py. Они включают в себя:

host: хост базы данных (по умолчанию localhost)
db_name: имя базы данных (по умолчанию my_db)
db_password: пароль базы данных (по умолчанию 12345)
db_user: пользователь базы данных (по умолчанию postgres)
port: порт базы данных (по умолчанию 5432)


## Использование

Убедитесь, что модуль psycopg2 установлен и доступен.
Создайте файл config_db.py с конфигурационными переменными.
Запустите скрипт.



## Функциональность
Скрипт выполняет следующие действия:

Проверяет доступность модуля psycopg2.
Подключается к базе данных с помощью конфигурационных переменных.
Создает таблицу list_users если она не существует.
Вставляет данные в таблицу list_users.
Закрывает соединение с базой данных.
Проверяет и меняет уровень изоляции.
Фиксация изменений - автоматически, при ошибке - откат.


## Функциональность
Скрипт выводит все действия, которые производит поочередно в консоль, обрабатывает ошибки при подключении к базе данных, создании таблицы и вставке данных. Если возникнет ошибка, она будет выведена в консоль.
